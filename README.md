# CISP401 Final - Reminder App
Reminder app written in Java using MVC structure
Can be used for creating, viewing, and deleting tasks.
The reminders are stored in a file.
# Features
* Add task with description, details, and due date
* Remove task
* View all saved tasks
* (Automatic) Saves data to local file

## Project Structure
* Models- Task data and storage logic
* Viewers- User input and output handling
* Controllers- User actions

## Key Ideas
* MVC STRUCTURE 
* Java file handling
* List operations
